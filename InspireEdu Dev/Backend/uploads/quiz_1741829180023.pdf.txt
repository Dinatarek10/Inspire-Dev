**1. (Easy)**  
What is the result of the Python expression `"hi" + " " + "name" * 3`?  
A) `"hi name name name"`  
B) `"hi namenamename"`  
C) `"hinamename"`  
D) `"hi name3"`  
**Answer:** B) `"hi namenamename"`

**2. (Medium)**  
If a user inputs the value `3` when prompted by `num = int(input("Type a number... "))`, what will `print(5 * num)` output?  
A) `15`  
B) `33333`  
C) `"33333"`  
D) `TypeError`  
**Answer:** A) `15`

**3. (Hard)**  
Consider the code:  
```python  
mysum = 0  
for i in range(5, 11, 2):  
    mysum += i  
    if mysum == 5:  
        break  
print(mysum)  
```  
What is the final value of `mysum`?  
A) `5`  
B) `21`  
C) `7`  
D) `16`  
**Answer:** A) `5`
